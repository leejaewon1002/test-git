# AI 학습 플래너

예산과 보유 자료를 기반으로 레시피를 추천하는 웹사이트 프로토타입입니다.

> 작은 진전이 쌓이면 큰 실력이 됩니다. 할 일 계획부터 목표 추적, 맞춤 학습 추천까지 한 번에 관리하세요.

## 페이지 구성

- **메인**: index.html
- **플래너**: pages/planner.html
- **목표**: pages/goals.html
- **추천**: pages/recommendations.html

## 실행 방법

정적 페이지이므로 브라우저에서 index.html을 열면 됩니다.

VS Code Live Server 확장 사용 시 루트에서 index.html을 우클릭하고 "Live Server로 열기"를 선택하면 자동으로 브라우저에서 전체 링크가 정상 동작합니다.

## 구현된 기능

### 1. 스마트 일정 관리
- 과목, 날짜, 예상 시간, 우선순위를 기준으로 학습 일정 등록
- 과목별 색상 태그로 한눈에 구분 (수학/영어/코딩/과학/기타)
- 우선순위/태그/날짜 다중 필터로 필요한 일정만 조회
- 로컬 저장소에 자동 저장되어 새로고침 후에도 유지

### 2. 달력 뷰
- 월간 캘린더로 학습 일정 시각화
- 달력의 각 날짜에 해당 날의 총 학습시간과 일정 개수 표시
- 이전/다음 월 이동 버튼으로 월별 조회 가능
- 달력 날짜 클릭으로 해당 날짜 일정만 필터링

### 3. 목표 달성 시각화
- 주간/월간 학습 목표 설정 및 진행률 입력
- 프로그레스 바로 목표 달성 진행 상황을 시각적으로 표시
- 목표 데이터는 로컬 저장소에 유지

### 4. AI 학습 추천
- 현재 학습 수준(초급/중급/고급)과 가능 시간, 학습 방식 선택
- 학습 수준별 맞춤형 루틴 자동 생성
- 시간대별 카드로 구성된 타임테이블 형태로 제시
  - 워밍업(개념 복습)
  - 집중 학습(문제 풀이)
  - 정리(복습 및 오답)
- 학습 방식에 따른 추가 팁 제공

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인, 그래디언트, 애니메이션
- **JavaScript**: 바닐라 JS로 상태 관리 및 렌더링
- **Local Storage**: 사용자 데이터 로컬 저장

## 디자인 특징

- 모던 미니멀 디자인으로 학습에 집중하는 환경 제공
- 부드러운 그래디언트와 떠다니는 배경 오브 효과
- 모바일 반응형으로 다양한 기기 지원
- 부드러운 등장 애니메이션으로 사용자 경험 향상

## 주요 기능 흐름

**일정 관리**
1. 플래너 페이지에서 과목/날짜/시간/우선순위/태그 입력
2. 입력된 일정이 리스트에 추가되고 자동 저장
3. 필터로 원하는 일정만 확인 가능
4. 달력에서 각 날짜의 일정 현황 확인

**목표 추적**
1. 목표 페이지에서 주간/월간 목표 시간 설정
2. 달성한 시간 입력
3. 실시간 프로그레스 바로 진행 상황 표시

**학습 추천**
1. 현재 학습 상황 입력 (수준/가능 시간/학습 방식)
2. AI가 개인화된 3단계 루틴 생성
3. 시간대별 추천 활동 확인
